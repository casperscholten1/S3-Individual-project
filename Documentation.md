# User stories

 - Als gebruiker wil ik al mijn matches van de afgelopen maand kunnen
   inzien zodat ik een duidelijk overzicht kan krijgen over wat ik
   afgelopen maand heb gedaan
   
 - Als gebruiker wil ik van een hero kunnen zien hoe vaak deze wordt
   gepicked per rank zodat ik kan kijken of een bepaalde hero bij mij
   kan passen
   
 - Als gebruiker wil ik de winrate van een hero per rank kunnen zien
   zodat ik weet of deze held in mijn bracket goed is
   
 - Als gebruiker wil ik een lijst kunnen zien van alle heroes zodat ik
   een duidelijk overzicht heb van alle mogelijke opties
   
 - Als gebruiker wil ik in deze lijst van alle heroes kunnen zoeken
   zodat ik informatie kan krijgen over een specifieke hero
   
 - Als gebruiker wil ik de lijst van heroes kunnen filteren op bijv.
   Primary Attribute, Attack type, Role zodat ik op zoek kan gaan naar
   een nieuwe held die bij mij past
   
 - Als gebruiker wil ik op een match kunnen zoeken zodat ik informatie
   kan krijgen van een specifieke match
